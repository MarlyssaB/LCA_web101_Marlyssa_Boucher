4. As a browser parses HTML, it constructs an internal document tree representing all the elements to be displayed. It then matches elements to styles specified in various spreadsheets according to the standard CSS cascade, inheritance, and ordering rules. The CSS engine evaluates each rule from right to left, starting in the key (rightmost) selector and moving through each selector until it finds a match or, failing to find a match, discards the rule.

3. Screen readers are software applications that read out the words and elements on a computer screen. They are most often used by people who are visually impared, have a learning disability, or are illiterate. ARIA stands for Accessible Rich Internet Applications and is a way to make the internet more accessible to those with disabilities. It does this by changing how certain elements are read out by a screen reader so that the user better understands what is on the screen. For example, a checkbox without ARIA could be misidentified as a 'group'. With ARIA, it will be correctly identified as an unchecked or checked checkbox. Accessibility is important because it allows your site to be used by all users.
